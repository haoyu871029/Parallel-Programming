照著以上 spec 中提供的指令依序執行來安裝與重編譯 UCX

注意一開始是在根目錄執行 git clone 指令，而不是在自己創建的 hw4 資料夾中執行，而在 ../configure 指令完成後，hw4 資料夾中會被存放一些 dynamic link files

因此執行完全部指令後，根目錄下應該要有 UCX-lsalab 資料夾與 hw4 資料夾。
